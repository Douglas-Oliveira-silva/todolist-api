# To-do List API

A simple and organized To-do List API built with Spring Boot, Maven, and H2 database.

## Features
- Add, view, update, and delete tasks
- In-memory database (H2)
- Spring Boot 3.2+
- Java 17+

## Technologies
- Java
- Spring Boot
- Maven
- H2 Database
- Lombok
